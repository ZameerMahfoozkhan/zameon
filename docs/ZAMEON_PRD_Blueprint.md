# ZAMEON --- Product Requirements Document (PRD)

## Vision

Build ZAMEON into a premium, globally recognizable ecommerce brand with
a shopping experience that feels handcrafted, elegant, fast, and
trustworthy. The website should not resemble a typical AI-generated
ecommerce template. Every section should have clear visual hierarchy,
intentional spacing, and premium interactions.

------------------------------------------------------------------------

# Brand Principles

-   Minimal but not empty
-   Premium but approachable
-   Fast over flashy
-   Product-first storytelling
-   Consistent visual language
-   Mobile-first responsive design
-   Accessibility by default

# Design System

## Grid

-   12-column desktop
-   8-column tablet
-   4-column mobile

## Spacing

Use an 8px spacing system throughout.

## Typography

-   Large display headings
-   Clean geometric sans-serif
-   Comfortable reading widths

## Components

Buttons, cards, chips, inputs, accordions, tabs, badges, drawers,
modals, breadcrumbs, search, mega menu, product cards, review cards,
newsletter block, trust badges.

------------------------------------------------------------------------

# Image Strategy

Do NOT generate embedded AI artwork inside the layout.

Every major section should contain professional placeholders labeled:

-   Hero Image Placeholder
-   Category Banner Placeholder
-   Lifestyle Image Placeholder
-   Product Gallery Placeholder
-   Team Image Placeholder
-   About Banner Placeholder
-   Newsletter Illustration Placeholder

Maintain proper aspect ratios.

# AI Development Instructions

The AI should build the website as if it were designed by a senior
product team.

Avoid: - Template look - Repetitive cards - Generic gradients -
Placeholder lorem ipsum - Oversized rounded corners - Crowded layouts

Favor: - Elegant whitespace - Strong typography - Premium interactions -
Reusable components - Clean code - Semantic HTML - Responsive layouts

Deliver production-quality structure with image placeholders instead of
generated artwork.
