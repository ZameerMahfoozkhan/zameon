# ZAMEON Ecommerce Website Master Prompt

## Project Overview

Build a **premium, world-class ecommerce website** for **ZAMEON**. This
must **not** resemble a generic AI-generated ecommerce template. The
final result should feel like it was crafted by a senior product design
team over several months.

### Brand Positioning

ZAMEON is a modern premium ecommerce brand that can expand beyond
gadgets into lifestyle, home, fashion, accessories, AI products, and
future categories.

### Primary Goal

Design a conversion-focused, lightning-fast, highly polished ecommerce
experience with exceptional UI/UX, micro-interactions, accessibility,
responsiveness, and SEO.

------------------------------------------------------------------------

# Critical Design Rules

-   Avoid cookie-cutter layouts.
-   Avoid oversized rounded cards everywhere.
-   Avoid random gradients.
-   Avoid generic ecommerce sections.
-   Maintain an 8px spacing system.
-   Use a professional design system.
-   Plenty of whitespace.
-   Smooth premium animations.
-   Pixel-perfect alignment.
-   Mobile-first.
-   WCAG-friendly.
-   Premium typography hierarchy.
-   Consistent border radius.
-   Modern glass effects only where appropriate.
-   Subtle shadows.

The website should feel inspired by Apple, Nothing, Linear, Stripe,
Vercel, Arc Browser and modern luxury ecommerce brands---not copied.

------------------------------------------------------------------------

# Temporary Branding

Generate a temporary minimalist logo using the word:

**ZAMEON**

Requirements: - Modern geometric sans-serif - Black monochrome - Simple
abstract "Z" icon - SVG-style appearance - Replaceable later - Include
favicon generated from the icon

------------------------------------------------------------------------

# Image Strategy

Do NOT generate AI artwork inside the page.

Instead, create clearly labeled image placeholders throughout the
design.

Examples: - Hero Product Image - Category Banner - Lifestyle Banner -
Product Thumbnail - Product Gallery - About Image - Newsletter
Illustration

Every placeholder should preserve the intended layout and aspect ratio.
